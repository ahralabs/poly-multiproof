Polynomial Multiproof Spec

3. Method 1

This method biases towards a faster opening time, and requires roughly half the compute of FastVerify. See A. Optimizations for more details.

For the following, the notation of BDFG21 is followed, namely

  • [k]={1,…,k}
  • For a set of points (x1​,…,xk​) and a polynomial fi​, ri​ is the unique degree k−1 polynomial such that ∀jri​(xj​)=fi​(xj​)

M1Open

Let the polynomials be (f1​,…,ft​) and the points be (x1​,…xk​)

  1. Check that evals has length t and each element of evals has length k
  2. Transcribes the following to the Merlin transcript
    • each of the evals row by row with the utf-8 message open evals
    • each of the points in order with the utf-8 message open points
  3. Constructs γ∈F by
    • Reading SCALAR_SIZE bytes from transcript with utf-8 message open gamma
    • Intepreting the bytes as an integer in big endian modded by the modulus of the scalar field
  4. Computing fsum​=i∈[t]∑​γi−1fi​
  5. Polynomial divide to get q=fsum​/Zpoint_set_index​, discarding the remainer
  6. Compute [q(x)]1​ and serialize in compressed form.

M1Verify

M1Verify(transcript: Transcript,

Let the evals be ((y1,1​,…y1,k​),…(yt,1​,…yt,k​)), commits be (c1​,…,ct​), and the points be (z1​,…zk​). Commit ci​ must be for the polynomial that evaluates to (yi,1​,…,yi,k​) at points (z1​,…,zk​).

This method

  1. Checks that each element has the correct length
    • Commits has length t
    • Each element of evals has length k
  2. Transcribes the points/evals the same as in the opening
  3. Reads γ same is in the opening
  4. For each j∈[k], computes aj​=i∈[t]∑​γj−1yi,j​ aj​ represents the value of ∑i∈[t]​γi−1ri​(zj​)
  5. Use the aj​ to interpolate ϕ=∑i∈[t]​γi−1ri​ using the previously computed lagrange polynomials li,j​ as ϕ=i∈[t]∑​γi−1ri​=j∈[k]∑​aj​lpoint_set_index,j​
  6. Compute α=[ϕ(x)]1​
  7. Compute β=∑i∈[t]​γi−1ci​
  8. Return true if e(β−α,g2​)=e(proof,θpoint_set_index​)
